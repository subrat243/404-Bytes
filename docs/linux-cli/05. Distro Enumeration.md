##### System Enumeration :- #####

>**lsb_release -a** = Use to know about distribution.

>**cat /etc/* release** = Use for distro info.

![[enum.png]]

>**lscpu** = Show info about CPU.

>**who** = Use to know the current User

>**groups** = Use to know the group of user.

```Copy
groups root
```
 
>**uname** = Unixname utility print system info or about kernal.

```Copy
uname -a 
```
**Here :-**

-a = For all
-s = To know kernal
-r = For kernal version
-p = Processor instruction set
-i = Hardware platform
-o = Operating system
