>**tar** - It's a command line utility which is used to create .tar archive.
### For compressing

```Copy
tar -cf file.tar File/
```

**Here :-**
-c = It is use to indicate to compress the file.
-f = It is use to indicate the file or archive file name.

```Copy
tar -cvf file.tar File/
```

-v = It is used for verbose
### For extracting

```Copy
tar -xvf file.tar
```

-x = It is use to indicate to extract the file.

>**gzip** - It's a command line utility which is used to create .tar.gz archive.

### creating .tar.gz archive file

```Copy
tar -czf file.tar.gz File/
```

-z = It used to indicate to compress the file in gzip format.

### For extracting

```Copy
tar -xzf file.tar.gz
```
