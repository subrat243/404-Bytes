
>Shell is **a text-based interface wherein users can enter command names, execute programs, and manipulate files such as file handles, file permissions, and directories**. Users can access the shell through the terminal window or console, a window displaying a command prompt where users can enter commands.

>**To check the shell :-**

```Copy
echo $SHELL
```

![[Check shell.png]]

>**To display current present shells :-** 

```Copy
cat /etc/shells
```

![[valid login shells.png]]

>**We can switch between shells using shell name in terminal like :-**

    sh, bash, dash etc

>**We can change the login shell in few steps :-**

     step 1 :- 

         Enter the command chsh in terminal.

     step 2 :- 

         Then enter the password.

     step 3 :-

         Then enter the /path of shell and hit enter.
