### What is a Shell?

A **shell** in Linux is a command-line interface (CLI) that allows users to interact with the system using commands. It acts as an interpreter between the user and the OS.

---

### Check Current Shell

```bash
echo $SHELL
```

**Example Output**:

```bash
/bin/bash
```

This shows your **current active shell**.

---

### List All Available Shells

```bash
cat /etc/shells
```

**Example Output**:

```bash
/bin/bash
/usr/bin/zsh
/usr/bin/fish
```

Shows all **installed shells** on the system.

---

### Switch Shell Temporarily

```bash
zsh
```

**To return**, just type:

```bash
exit
```

---

### Change Default Shell (Permanent)

```bash
chsh -s /usr/bin/zsh
```

You may be prompted for your password.

---

### Popular Shells Compared

|Feature|Bash|Fish|Zsh|
|---|---|---|---|
|Full Name|Bourne Again Shell|Friendly Interactive Shell|Z Shell|
|Scripting|Widely supported|Limited compared to Bash/Zsh|Advanced scripting with plugin support|
|Tab Completion|Basic|Intelligent and history-aware|Enhanced (with plugins)|
|Customization|Moderate|Interactive UI|Extensive (e.g., Oh My Zsh)|
|User Friendliness|CLI-focused|Most user-friendly|Depends on setup|
|Syntax Highlighting|No (by default)|Yes (built-in)|Yes (with plugins like `zsh-syntax-highlighting`)|

### What is a Shell Script?

A **shell script** is a text file containing a series of commands that the shell executes in sequence.

**File Extension**: `.sh`

---

### Creating a Script

#### 1. Create a file

```bash
nano first_script.sh
```

#### 2. Add shebang + code

```bash
#!/bin/bash
echo "Hello World"
```

#### 3. Save and Exit

- Press `CTRL+X`, then `Y`, then `ENTER`.
    

#### 4. Make executable

```bash
chmod +x first_script.sh
```

#### 5. Run the script

```bash
./first_script.sh
```

---

### Script Basics with Examples

---

#### Variables

```bash
#!/bin/bash
echo "What’s your name?"
read name
echo "Welcome, $name"
```

**Explanation**:

- `read name`: Takes user input and stores in `name`.
    
- `$name`: Prints the value stored in the variable.
    

---

#### Loops

##### `for` Loop (Print 1 to 10)

```bash
#!/bin/bash
for i in {1..10}
do
  echo $i
done
```

**Explanation**:

- `{1..10}`: Range of numbers.
    
- `$i`: Current value in the loop.
    

---

#### Conditional Statements

```bash
#!/bin/bash
echo "Enter your name:"
read name

if [ "$name" = "Stewart" ]; then
  echo "Welcome Stewart! Here is the secret: THM_Script"
else
  echo "Sorry! You are not authorized to access the secret."
fi
```

**Explanation**:

- `if [ "$name" = "Stewart" ]`: Checks if the input equals "Stewart".
    
- `then`: Executes if condition is true.
    
- `else`: Executes if condition is false.
    
- `fi`: Ends the conditional block.
    

---

### Comments

```bash
#!/bin/bash

# Asking for the user's name
echo "Enter your name:"
read name

# Checking if the user is Stewart
if [ "$name" = "Stewart" ]; then
  # Authorized user message
  echo "Welcome Stewart! Here is the secret: THM_Script"
else
  # Unauthorized user message
  echo "Sorry! You are not authorized to access the secret."
fi
```

**Good Practice**: Use comments to explain complex or critical logic blocks.
