# 13. Curl Fundamentals

>**curl - (Client URL) is a command-line tool that allows data transfer to or from a server without user interaction using the supported libcurl library. curl can also be used to troubleshoot connection issues.

>**Supported protocols:-

     http
     https
     ftp
     sftp
     telnet, etc.
#### Syntax:-

```Copy
curl <URL>
```

>**It will show the content of the website. Example:-
>
```Copy
curl https://www.campustocrypto.com
```

>**Saving the output of the query.

```Copy
curl -o /PATH <URL>
```

---
#### Downloading Files from web using curl:-

>**It will download the file with the given name and extension from the URL.

```Copy
curl -o <filename.extension> <URL>
```

>**It will directly download from the URL.

```Copy
curl -O <URL>
```

>**If a website is redirecting to another URL, it is very important to know how to specify the URL to curl

```Copy
curl -L <URL>
```

>**Viewing response header. It can give the response from website in the output.

```Copy
curl -I <URL>
```

>**Viewing request header, connection details and TLS handshake, etc.

```Copy
curl -v <URL>
```

>**Posting request on the website like login credentials and etc.

```Copy
curl --data "credentials and parameters" <URL>
```

Example:-

```Copy
curl -data "log=admin&pwd=passwd" https://www.facebook.com
```
