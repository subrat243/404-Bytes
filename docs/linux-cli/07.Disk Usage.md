>**du** - A command line utility used to estimate the file space usage in the system.

```Copy
du
```

![[du.png]]

>To summarize and show the contents in human readable form we use following command :-

```Copy
du -sh *
```

![[du -sh astrik.png]]

>Here -s is for summarize and -h is for human readable form.

>To show the total size usage we use the following command :-

```Copy
du -shc *
```
 
![[du -shc astrik.png]]

>Here -c indicates to show the total.

>**df** - Used to report file system disk space usage.

```Copy
df
```

![[df.png]]

>To show in human readable form we use the following command :-

```Copy
df -h
```

![[df -h.png]]

>To show the usage of space by file system we use the following command :-

```Copy
df -ht 'file system name'
```

![[df -ht ext4.png]]

>Ex :- ext4, ext6, etc.
