# 11. Services And Process Management

>**top** = Display linux process.

```Copy
top
```

>**htop** = Interactive process viewer.

```Copy
htop
```

>**free** = Display amount of free and used memory in the system.

```Copy
free -h
```

>**ps** = Report a snapshot of the current processes.

```Copy
ps aux
```

>**kill** = Terminate or stop the process.

```Copy
kill "PID"
```

##### System D

>systemctl =  Control the SystemD system and service manager.

```Copy
systemctl <ACTION> "Servicename"
```

Actions:-

     start   -> Starts the service
     stop    -> Stops the service
     status  -> Show service status
     restart -> Restart the service with same configuration
     reload  -> Reload the service with new configuration

Example:-  

```Copy
systemctl start tor
```

##### System Services On Startup

To show the service is enable for startup.

```Copy
systemctl is-enabled "servicename"
```

To enable the service for startup.

```Copy
systemctl enable "servicename"
```

To disable the service for startup.

```Copy
systemctl disable "servicename"
```

To reload or reconfigure the services if there is any issue.

```Copy
systemctl daemon-reload
```

##### System V

```Copy
service  "Servicename" <ACTION>
```

Actions:-

     start   -> Starts the service
     stop    -> Stops the service
     status  -> Show service status
     restart -> Restart the service
