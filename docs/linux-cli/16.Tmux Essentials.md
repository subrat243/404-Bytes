>**To initiat Tmux

```Copy
tmux
```

>**To split the terminal horizontally**

```
Type: ctrl + b  %
```

>**To split the terminal vertically**

```
Type: ctrl + b  "
```

>**To move between panes**

```
ctrl + b use direction key
```

>**Creating new windows**

```
ctrl + b c
```

>**Switching between windows**

```
ctrl + b 0,1,2 <according to the windows no. or name>
```

>**Rename windows**

```
ctrl + b ,
```

#### Creating sessions to preserve terminal's action
>**To detach the session**

```
ctrl + b d
```

>**To list the tmux sessions**

```Copy
tmux ls
```

>**To attach the session**

```Copy
tmux attach -t <no. or name of session>
```

>**Creating sessions with name**

```Copy
tmux new -s <name>
```

>**Renaming sessions**

```Copy
tmux rename-session -t 0 <name>
```
