# 01. File Management and Manipulation


>**pwd** = The present working directory command, return your location within the directory structure.

![[pwd.png]]

>**ls** = To see the contents of a directory (the files and subdirectories), we can use the ls (list) command. This is very similar to the dir command in Windows.

![[ls.png]]

==To get more information about the files and directories, such as their permissions, owner, size, and when they were last modified. we can use== :-

>**ls -l** = Provides us with significantly more information, such as whether an object is a file or directory, the number of links, the owner, the group, its size, when it was created or modified, and its name.

>**ls -a** = List all the files and directories including hidden files.

>**ls -la** = The combination of above command. It will list the  file or directory, the number of links, the owner, the group, its size, when it was created or modified, and its name  including hidden files.

![[ls -la.png]]

>**ls -lh** = List the content in human readable form where 'h' is human readable.

>**ls -lR /path or filename** = List the sub-directories of the given directory or path.

>**cd** = To change directories from the terminal, use the change directory command.

![[cd.png]]

>**cd ..** = To move up one level in the file structure (toward the root of the file structure, or /)

![[cd ...png]]

==you can move up as many levels as you need.==

>**Just use the same number of double dot pairs as the number of levels you want to move: 

- You would use .. to move up one level.

- You would use .. .. to move up two levels.

- You would use .. .. .. to move up three levels, and so on.

>**cd ~** = To navigate to the home directory from any directory.

>**cd /** = To move to the root directory directly.

>**whatis "name"** =  To know about utility, pkg or tools

![[whatis.png]]

>**touch "filename.txt"** = Use to create file.

![[touch.png]]

>**echo** = Use to display a line of text.

>**echo "text" > filename.txt** = Use to direct or input the data to the file.

![[echo.png]]

>**cat** = Prints the content of the file in output.

>**cat > filename** = Use to create file and to redirect the content to the file.

>**cat file1.txt or /path > file2.txt** = Use to redirect the content of the file to the another file.

>**cat >> filename** = Use to add more content to the file with double redirect (">>").

![[cat.png]]

>**file** = The `file` command gives us the type of data of the file. Some examples would be: ‘ELF’, ‘Perl script’, ‘ASCII text’, ‘data’ and more.

```Copy
file "filename"
```

>**For all files.**

```Copy
file ./*
```

>**mkdir** = Use to create directory.

```Copy
mkdir "dirname"
```

>**cp** = Use to copy file to the directory.

```Copy
cp "filename.txt" "dirname" or /path
```

>**mv** = Use to move file directory to directory. Also use to rename a file.

```Copy
mv "filename.txt" "dirname"
```
  
>**for renaming: 

```Copy
mv "file.txt" "newfile.txt"
```

>**rm "filename"** = Use to remove the file

>**rmdir** = Use to remove directory, but only empty once.

>**rm -R** = Use to remove directory with content where "R" means recursively.

>**rm -rf** = Also removes directory.
