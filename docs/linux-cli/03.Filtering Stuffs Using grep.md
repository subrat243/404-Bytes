
>**grep** = Search for string and pattern in the file or print lines matching a pattern.

```Copy
gerp "dynamic" /path or 'filename'
```

>Note :- Pattern to be find in " " .

>grep tool is case sensitive and using "-i " we can override it. 

```Copy
grep -i "dynamic" /path or 'filename'
```

"-i " is used to find all word or string in file related to given string.

>The grep command is often used when output is piped from one command to another. Linux (and Windows for that matter) allows us to take the output of one command and send it as input to another command. This is called piping, and we use the " | " command to do it.

```Copy
cat /path or 'filename' | grep "string"
```
#### Using grep with locate :- ####

>We can use locate command's output to pipe in grep to capture the file or stuffs, etc.

```Copy
locate 'filename' | grep /path or 'filename'
```

>And so on.
