>**netstat** - Print network connections, routing tables, interface statistics, masquerade connections, and multicast memberships

```Copy
netstat 'option'
```

![[netstat.png]]

-r = Display the kernel routing tables. netstat -r and route -e produce the same output.
-i = Display a table of all network interfaces.
-s = Display summary statistics for each protocol.
-v = Tell the user what is going on by being verbose. Especially print some useful information about unconfigured address families.
-p = Show the PID and name of the program to which each socket belongs.
-t = Show TCP connections.
-u = Show UDP connections.
-l = Show only listening sockets.

 >**[[Netdiscover]]** - An active/passive ARP reconnaissance tool, initially developed to gain information about wireless networks without DHCP servers in wardriving scenarios. It can also be used on switched networks. Built on top of libnet and libpcap, it can passively detect online hosts or search for them by sending ARP requests. Furthermore, it can be used to inspect your network's ARP traffic, or find network addresses using auto scan mode, which will scan for common local networks.
 
```Copy
netdiscover -i 'interface'
```
